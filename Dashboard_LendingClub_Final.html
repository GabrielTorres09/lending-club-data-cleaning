Objetivo del proyecto

¿Qué queremos lograr?

Objetivo principal

Preparar, limpiar y analizar el dataset de Lending Club (2007–2018) para entender el comportamiento de préstamos de consumo en EE. UU., como base para futuros modelos predictivos de riesgo crediticio.

Dataset utilizado

2.26 millones de préstamos aprobados entre 2007 y 2018, descargados desde Kaggle mediante API. Incluye variables financieras, crediticias y demográficas de cada solicitante.

Proyección futura

Con el dataset limpio se buscará entrenar modelos de clasificación (regresión logística, árbol de decisión o random forest) para predecir la probabilidad de impago de un préstamo.

Problema identificado

¿Qué desafío de datos resolvimos?

Alta tasa de valores nulos

El 57% de las columnas (87 de 151) tenían valores faltantes. El 50% de ellas superaba el umbral del 50% de nulos, haciendo imposible su uso directo en modelos.

Dimensionalidad excesiva

El dataset original contaba con 151 columnas, muchas redundantes o con información de baja calidad. Se redujo a 107 columnas manteniendo toda la información relevante.

Nomenclatura en inglés

Todas las columnas estaban en inglés técnico financiero. Se tradujo y documentó cada columna al español con su descripción para facilitar el análisis y la presentación.

Métodos y técnicas aplicadas

Proceso metodológico de limpieza y preparación de datos

Obtención de datos

Descarga automatizada desde Kaggle API con Python (kaggle datasets download). Dataset en formato CSV.

Eliminación de columnas

Se eliminaron 44 columnas con más del 50% de valores nulos usando df.dropna(thresh=...) y análisis de nulidad por columna.

Imputación

29 columnas numéricas imputadas con mediana. 14 columnas de texto imputadas con moda. Estrategia según tipo de dato.

Deduplicación

Verificación de duplicados con df.duplicated() (exactos) y subset=['id'] (por ID). Resultado: 0 duplicados.

Análisis exploratorio

EDA con Pandas y visualizaciones. Estadísticas descriptivas (media, mediana, percentiles) de montos, tasas e ingresos. Distribuciones por estado y propósito.

Traducción y documentación

Las 107 columnas finales fueron renombradas al español con descripciones claras, organizadas en 5 categorías temáticas para facilitar su uso.

Ética, privacidad y limitaciones

Consideraciones responsables sobre el uso de datos financieros

Sesgos potenciales del dataset

El dataset puede contener sesgos históricos de aprobación de crédito basados en variables como estado de residencia, ingreso o tipo de empleo. Los modelos entrenados en estos datos podrían reproducir discriminación sistémica si no se controlan adecuadamente.

Datos sensibles

El dataset incluye información financiera personal (ingresos, deudas, historial crediticio). Este proyecto usa datos anonimizados de acceso público en Kaggle, sin identificación individual de personas.

Licencia y fuente

Datos de acceso público en Kaggle bajo uso académico/investigativo. El dataset fue publicado originalmente por Lending Club. Uso exclusivamente educativo en el contexto de la UTB.

Limitaciones del análisis

Los datos cubren 2007–2018 y no reflejan el mercado crediticio actual. Las conclusiones son descriptivas del periodo analizado. Los modelos futuros deben validarse con datos recientes antes de cualquier aplicación real.

Compromiso ético: Este proyecto tiene fines exclusivamente académicos. No se utilizan datos de personas identificables, no se toman decisiones crediticias reales con estos resultados, y se reconoce que cualquier modelo predictivo derivado debe evaluarse con criterios de equidad antes de su aplicación.
